# symphony_api

Environnement

This app runs on:

    PHP 7.6

Installation

    Run composer selfupdate
    Run composer install

Utilization


/user/get/{id} GET

Create new user account
Parameters :

    name (REQUIRED) : your account name
    
Get user by id
/user/create/{id} GET

Delete a user
/user/delete/{id} DELETE

Disconnect the user from his session
